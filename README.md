# IT 资产管理维护系统

  对公司的 IT 资产进行持续的跟踪管理。
  
  基于 ThinkPHP6 + Layui

  包含模块：

  * 管理员
  * 部门
  * 员工
  * 软件
  * 电脑
  * 打印机
  * 维护记录
